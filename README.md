# AWD Server
ミニ四駆イベント用 スコア管理サーバ

## Require
- nodenv

## Installation
```
$ git clone https://github.com/tdu-naklab/awd_server.git
$ cd awd_server
$ nodenv install
$ npm install
$ npm start
```
